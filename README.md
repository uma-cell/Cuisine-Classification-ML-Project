# 🍽️ Cuisine Classification using Machine Learning

## 📌 Project Overview
This project is part of my internship at **Cognifyz IT Solutions**.  
The goal is to build a Machine Learning model that classifies restaurants based on their cuisines using various restaurant features like ratings, votes, cost, and location.

---

## 🚀 Project Title
**Cuisine Classification using Machine Learning (Random Forest Classifier)**

---

## 🎯 Objective
To predict the cuisine type of a restaurant based on input features using supervised machine learning techniques.

---

## 📊 Dataset
- Dataset contains restaurant information
- Features include:
  - Rating
  - Votes
  - Cost
  - Location
  - Cuisines (Target column)
